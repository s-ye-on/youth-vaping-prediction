# 최종 보고서 요약

## 프로젝트 목적
청소년 건강행태조사 2023~2025 통합 데이터를 이용해 액상형 전자담배 현재 사용 위험 요인을 예측하고, SHAP으로 주요 영향 피처를 해석한다.

## 데이터 구성
입력 데이터는 개인 단위 설문 응답과 feature catalog로 구성되며, 전처리 후 모델 학습용 데이터로 변환했다.

## 모델 성능 요약
최고 성능 모델은 `random_forest`입니다. 위험군 탐지가 중요하므로 Accuracy보다 Recall, F1-score, ROC-AUC를 함께 확인했습니다.

## 주요 영향 피처
- `TC_LT`: 평균 절대 SHAP 0.0885
- `TC_HTP_LT`: 평균 절대 SHAP 0.0508
- `AC_LT`: 평균 절대 SHAP 0.0504
- `TC_ACCESS`: 평균 절대 SHAP 0.0494
- `AC_ACCESS`: 평균 절대 SHAP 0.0292
- `S_SI`: 평균 절대 SHAP 0.0198
- `INT_SPWD_TM`: 평균 절대 SHAP 0.0148
- `INT_SPWK_TM`: 평균 절대 SHAP 0.0132
- `TC_SND_P`: 평균 절대 SHAP 0.0121
- `M_SAD`: 평균 절대 SHAP 0.0098

## SHAP 해석
SHAP 그룹 기준으로는 `흡연 관련` 그룹의 중요도 합계가 가장 큽니다.

## 시뮬레이션 결과
원본 평균 예측 확률은 0.1762이며, 가장 낮은 평균 예측 확률을 보인 시나리오는 Scenario 4(0.1705)입니다.
단, 이 시뮬레이션은 실제 정책 효과가 아니라 모델 기반 가상 변화입니다.

## 결론
모델은 청소년 액상형 전자담배 사용 위험군을 구분하는 데 활용 가능하며, SHAP 결과는 주요 영향 피처를 설명하는 근거로 사용할 수 있습니다.

## 한계 및 향후 개선
3개년 설문 데이터만 사용했기 때문에 장기 추세와 외부 환경 요인은 충분히 반영하지 못했습니다. 향후 지역별 정책, 가격, 접근성 데이터를 결합하고 시계열 예측 방식을 고도화할 필요가 있습니다.
